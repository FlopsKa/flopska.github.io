On High-Contrast Subspaces and Monte Carlo Dependency Estimation

Posted on: Do 13 August 2020

Estimating dependencies of random variables is of fundamental interest in statistics. Accordingly, lots of different measures quantifying the bivariate or multivariate connections between these functions are available. HiCS and MCDE are two approaches at solving this problem in the bi- and multivariate case while also adressing the problems arising with data of high dimensionality.

Introduction

Definitions

Conditional Probability

If EEE is an event with positive probability, the conditional probability of the event AAA, given EEE, written P(A∣E)P(A|E)P(A∣E), is defined by the relation

P(A∣E)=P(AE)P(E).P(A|E) = \frac{P(AE)}{P(E)}.P(A∣E)=P(E)P(AE)​.

Notion of Independence

Two events AAA and BBB are said to be independent if and only if the following product rule holds:

P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)

A class of events A\mathcal AA is said to be an independent class if and only if the product rule holds for every finite subclass of A\mathcal AA.

Random variables X(⋅)X(\cdot)X(⋅) and Y(⋅)Y(\cdot)Y(⋅) are independent if and only if

PXY(M×N)=PX(M)PY(N)P_{XY}(M \times N) = P_X(M)P_Y(N)PXY​(M×N)=PX​(M)PY​(N)

for each Borel set MMM in R1R_1R1​ and NNN in R2R_2R2​.

Equivalently, we can state:

Two random variables X(⋅)X(\cdot)X(⋅) and Y(⋅)Y(\cdot)Y(⋅) are independent if and only if their distribution functions satisfy the product rule FXY(t,u)=FX(t)FY(u)F_{XY}(t,u) = F_X(t)F_Y(u)FXY​(t,u)=FX​(t)FY​(u).

General framework

Independence Assumption

If we assume X\mathbf XX is an independent subspace, ∏Xip(Xi)=p(X)\prod_{X_i} p(X_i) = p(\mathbf X)∏Xi​​p(Xi​)=p(X) must hold. We know that if XXX is independent of YYY we have p(X∣Y)=p(X)p(X|Y) = p(X)p(X∣Y)=p(X). Hence ∏i,ji≠jp(Xi∣Xj)=p(X)\prod_{i,j\\i\neq j} p(X_i|X_j) = p(\mathbf X)∏i,ji​=j​p(Xi​∣Xj​)=p(X). It is clear that p(Xi∣Xj)=p(Xi)p(X_i|X_j) = p(X_i)p(Xi​∣Xj​)=p(Xi​) for all i,ji,ji,j and i≠ji \neq ji​=j is a necessary condition for an independent subspace.

Dependency of marginal distributions

Given that p(Xi∣Xj)=p(Xi)p(X_i|X_j) = p(X_i)p(Xi​∣Xj​)=p(Xi​) is a necessary condition we measure how "far away" we are from fulfilling this condition. MCDE and HiCS suggest using a statistical test to quantify this violation of independence and term this measure "contrast".

We define the discrepancy DDD as the difference of a marginal distribution p(Xi)p(X_i)p(Xi​) to the same distribution conditioned on another variable p(Xi∣Xj)p(X_i|X_j)p(Xi​∣Xj​). One possibility to quantify the difference of two distributions is using the Kolmogorov-Smirnov-Test of the respective cumulative distribution functions F(⋅)F(\cdot)F(⋅):

Dj0=sup⁡∣F(Xi)−F(Xi∣Xj=xj0)∣ D_{j_0} = \sup | F(X_i) - F(X_i|X_j=x_{j_0})| Dj0​​=sup∣F(Xi​)−F(Xi​∣Xj​=xj0​​)∣

Integrating over x_j this gives

D=∫XjDxjdxj. D = \int_{X_j} D_{x_j} dx_j. D=∫Xj​​Dxj​​dxj​.

Proposal

Working with sup⁡\supsup and absolute values is not so nice. We propose using the squared pointwise distance instead, like in the Cramer-von-Mises Test:

Dj0=∫Xi[F(xi)−F(xi∣Xj=xj0)]2dxi D_{j_0} = \int_{{X_i}} [F(x_i) - F(x_i | X_j = x_{j_0})]^2 dx_i Dj0​​=∫Xi​​[F(xi​)−F(xi​∣Xj​=xj0​​)]2dxi​

This gives

D=∫Xj∫Xi[F(xi)−F(xi∣Xj=xj0)]2dxidxj=∫Xj∫XiF(xi)2−2F(xi)F(xi∣Xj=xj0)+F(xi∣Xj=xj0)2dxidxj. \begin{aligned} D &= \int_{X_j} \int_{{X_i}} [F(x_i) - F(x_i | X_j = x_{j_0})]^2 dx_idx_j \\ &= \int_{X_j} \int_{{X_i}} F(x_i)^2 - 2F(x_i)F(x_i | X_j = x_{j_0}) + F(x_i | X_j = x_{j_0})^2 dx_idx_j. \end{aligned} D​=∫Xj​​∫Xi​​[F(xi​)−F(xi​∣Xj​=xj0​​)]2dxi​dxj​=∫Xj​​∫Xi​​F(xi​)2−2F(xi​)F(xi​∣Xj​=xj0​​)+F(xi​∣Xj​=xj0​​)2dxi​dxj​.​

Examples:

  1. Taking X_j,X_i independent implies D=0D=0D=0.
  2. taking X_i uniform with X_j=X_i

We would like to find some linear transformation VVV of XXX such that DDD is maximal. This corresponds to PCA but instead of maximizing the variance we maximize the contrast function DDD.
